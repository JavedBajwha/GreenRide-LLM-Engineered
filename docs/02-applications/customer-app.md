# Customer App

## Purpose

The Customer App is the passenger-facing application layer of GreenRide.

It allows customers to:
- search journeys
- get quotes
- create bookings
- track trips
- manage account

---

## Core Capabilities

### Booking
- search & quote
- booking creation
- guest checkout
- account booking

### Account
- profile
- history
- saved locations

### Tracking
- driver details
- live location

---

## Requirements

- fast booking flow
- minimal friction
- mobile friendly
- tenant branding support

---

## Rules

- must align with booking flow
- must use API contracts
- must not invent behaviour

---

## Notes

Customer App is a core entry point for revenue.
It must be simple, fast, and reliable.
