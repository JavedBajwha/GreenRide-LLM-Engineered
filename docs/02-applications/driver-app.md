# Driver App

This file is the stable navigation target for `[[driver-app]]` references.

The detailed driver application specification lives in:

- `driver-app-full.md`

## Summary

The Driver App is the operational app used by drivers to:

- receive jobs
- update trip status
- share realtime location when enabled
- complete the trip lifecycle safely

## Rule

Use `driver-app-full.md` as the detailed source of truth for driver workflows.
Do not invent additional driver states or screens outside the documented scope.
