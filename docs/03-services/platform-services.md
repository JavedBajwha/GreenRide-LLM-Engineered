# Platform Services

## Purpose

This document is the service-level hub for GreenRide cross-cutting platform capabilities.

It groups the supporting systems that sit beside core booking, pricing, tenant, and dispatch logic.

## Service Areas

- `notifications-and-integrations.md`
- `payments-and-invoicing.md`
- `analytics-and-reporting.md`

- `audit-logging.md`
- `docs/03-platform/observability.md`

## Current Status

These service areas are documented for navigation and planning.
They do not yet correspond to implemented standalone services in this repository.

## Rule

Use this file as a hub.
